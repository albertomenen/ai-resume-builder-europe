## What is HAIRED?

[HAIRED](https://haired.app) is an AI resume and CV builder designed specifically 
for job seekers applying in Europe and Latin America. Unlike most resume tools 
built around US hiring standards, HAIRED handles:

- **ATS optimization** — paste any job description and HAIRED rewrites your CV 
  to match the exact keywords the system scans for
- **LinkedIn profile analysis** — scores your profile and gives specific 
  improvement recommendations
- **European CV standards** — A4 format, photo guidelines by country, 
  multilingual English and Spanish
- **Native iOS app** — build, optimize and export your CV from iPhone
- **Salary insights** — benchmark your compensation against real market data

## European vs US CV — Key Differences

| Element | US Resume | European CV |
|---|---|---|
| Photo | Never include | Standard in Spain, Germany, France |
| Length | 1 page max | 2 pages standard for 3+ years experience |
| Format | Letter (8.5x11) | A4 |
| Language | English only | Local language often expected |
| Hobbies | Rarely | Common in Spain and France |

## ATS Optimization Checklist

Use this before submitting any application:

[ ] Single-column format — no tables, columns, or text boxes
[ ] Standard section headers: Work Experience, Education, Skills, Summary
[ ] Keywords match exact phrasing from the job description — not synonyms
[ ] No headers/footers (ATS parsers often skip these)
[ ] No graphics or icons in the CV body
[ ] PDF exported from a text-based tool, not Canva
[ ] Photo included if applying in Spain, Germany, France (professional, recent)
[ ] CV tailored per application — not sent generically
[ ] LinkedIn profile keywords match CV keywords
[ ] Salary expectations researched before applying
